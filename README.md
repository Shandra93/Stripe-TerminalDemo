1st Step:
Go to your terminal on visualStudio.
Git clone the URL on the terminal from VisualStudio: https://github.com/Shandra93/Stripe-TerminalDemo/tree/main

2nd Step:
Get the Api secret key from your Stripe Dashboard

3rd Step:
Create a .env file on the main TerminalDemo Folder.

5th Step:
Run npm install to install the node libraries

6th Step:
To Run the Demo, type npm install in the terminal.

7th Step:
If everything is set properly, it should be running smoothly.
